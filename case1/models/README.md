# Models Directory

Large model files are not included in this GitHub repository because GitHub blocks files larger than 100MB.

Before running the project, copy the required model files into this directory.

Required files may include:

- face_detection.om
- face_recognition.om
- 1k3d68.onnx
- 2d106det.onnx
- det_500m.onnx
- genderage.onnx
- w600k_mbf.onnx
